There are about 40 different specialties in the data. The categories are not evenly distributed: some have many records (e.g., Surgery), while others have very few (e.g., Hospice – Palliative Care). Some categories are also related or overlapping, such as Neurology and Neurosurgery.

#question 2
mt_samples %>%
  unnest_tokens(word, transcription) %>%
  count(word, sort = TRUE) %>%
  top_n(20, n) %>%
  ggplot(aes(x = reorder(word, n), y = n)) +
  geom_col()+
  coord_flip()+
  labs(x = "Word", y = "Count", title = "Top 20 Frequent Words")

Most of the top words are common stopwords like the, and, was, of, to. That’s expected since they appear very often in everyday language. One word that stands out is patient, which makes sense given the medical context.

#question 3
mt_samples %>%
  unnest_tokens(word, transcription) %>%
  anti_join(stop_words %>% filter(word %in% c("right", "left")), by = "word") %>%
  filter(!grepl("^[0-9]+$", word)) %>%
  count(word, sort = TRUE) %>%
  top_n(20, n) %>%
  ggplot(aes(x = reorder(word, n), y = n)) +
  geom_col()+
  coord_flip()+
  labs(x = "Word", y = "Count", title = "Top 20 Words ( Excluding Stop Words & Numbers)")

Now that stop words and numbers are removed, the top words are things like patient, left, history, procedure, pain. This gives a much clearer picture that the text is about medical notes and clinical procedures, instead of being dominated by common words.

With bigrams, we see common pairs like the patient, of the, in the. When we move to trigrams, phrases become more specific, like the patient was, the operating room, at this time. So bigrams still show a lot of stopword pairs, while trigrams start to capture more meaningful clinical phrases.

top_specialties <- mt_samples %>%
  count(medical_specialty, sort = TRUE) %>%
  slice_max(n, n = 4) %>%
  pull(medical_specialty)

mt_samples %>%
  filter(medical_specialty %in% top_specialties) %>%    
  unnest_tokens(word, transcription) %>%
  anti_join(stop_words, by = "word") %>%
  filter(!grepl("^[0-9]+$", word)) %>%
  count(medical_specialty, word, sort = TRUE) %>%
  group_by(medical_specialty) %>%
  slice_max(n, n = 10, with_ties = FALSE) %>%
  ungroup() %>%
  mutate(word = reorder_within(word, n, medical_specialty)) %>%
  ggplot(aes(n, word, fill = medical_specialty)) +
  geom_col() +
  scale_y_reordered() +
  facet_wrap(~ medical_specialty, scales = "free_y") +
  labs(x = "Count", y = "Word", title = "Top 10 Words by Specialty") +
  theme(legend.position = "none")
